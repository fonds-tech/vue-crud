# context-menu 模块

## 目的
提供右键菜单能力及渲染辅助，支持嵌套项与快捷键提示。

## 模块概述
- **职责:** 菜单项解析、坐标定位、渲染与关闭交互
- **状态:** 🚧开发中
- **最后更新:** 2025-12-11

## 规范

### 需求: 覆盖率提升
**模块:** context-menu
补测 helpers/hooks 边界分支与菜单渲染透传属性。

#### 场景: 渲染与事件
- 非 props attrs 透传过滤行为
- 空/异常菜单项处理

## API接口
无 HTTP API，提供组件与工具函数。

## 数据模型
无持久化数据模型。

## 依赖
- Element Plus
- context-menu/core 与 render 子模块
