# table 模块

## 目的
提供可配置的表格组件与核心逻辑，支持列设置、分页、工具栏、操作列等。

## 模块概述
- **职责:** 表格渲染、列配置、排序筛选、工具栏动作、表格状态管理
- **状态:** 🚧开发中
- **最后更新:** 2025-12-11

## 规范

### 需求: 覆盖率提升
**模块:** table
补充分支与边界测试，覆盖列配置重建、工具栏行为、渲染分支。

#### 场景: 渲染与交互边界
- 验证设置弹窗属性透传/非透传处理
- 验证工具栏显示/隐藏分支
- 验证渲染高度/尺寸调整分支

## API接口
当前无独立 HTTP API，对外通过组件 props/事件。

## 数据模型
无持久化数据模型。

## 依赖
- Element Plus 组件库
- table/core 与 render 子模块
